<h2>状态码</h2>



<h4>1. 响应状态码</h4>

> * 可根据具体情况与技术沟通添加新的响应状态码

| 错误代码 | error描述                                        |
| :------- | :----------------------------------------------- |
| 200      | 成功、已付款、发放成功                           |
| 201      | 未付款、发放失败                                 |
| 202      | userId、serverId、roleId不存在                   |
| 203      | 当前用户已购买周卡、月卡、极卡、年卡无需重复下单 |
| 204      | 订单已发放                                       |
| 205      | 签名错误                                         |
| 206      | 订单初始化失败                                   |
| 500      | 系统错误                                         |



<h4>2. 交易状态码</h4>

| 状态码     | 描述                |
| :--------- | :------------------ |
| SUCCESS    | 付款成功            |
| REFUND     | 退款（由商户发起）  |
| CHARGEBACK | 退款（ 由玩家发起） |
| REVERSED   | 拒绝退款            |
